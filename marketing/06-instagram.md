**하이라이트 5개** (커버: #FF5A1F 단색 + 흰 라인 아이콘, 이름 4자 이내): 신청방법(탭 아이콘) / 이번주(달력) / 후기(말풍선) / 코스(핀) / FAQ(물음표)

## 2. 첫 4주 그리드 (주 3회: 월/목/토)

타입: **O**(오렌지 카드뉴스) / **P**(사진 — 웜톤 통일, 저녁·야경만) / **Q**(흰 배경+오렌지 타이포 인용카드)

```
[O 후기]  [P 현장]  [Q 인용]   4주차
[P 현장]  [O 가이드] [Q 인용]   3주차
[O 무료]  [Q 인용]  [P 코스]   2주차
[Q 선언]  [P 첫현장] [O 인트로]  1주차
```

| 주 | 월 | 목 | 토 |
|---|---|---|---|
| 1 | O 인트로 세트 | P 첫 현장(없으면 duo_sunset) | Q 브랜드 선언 |
| 2 | O 무료화 세트 | Q 1주차 참가자 인용 | P 코스(hangang_night) |
| 3 | P 발맞춤(feet_sync) | O 가이드 세트 | Q 인용 |
| 4 | Q 참가자 인용 | P 현장 단체 | O 후기 세트(실후기 재편집) |

**그리드 헌법**: ①오렌지 블록 인접 금지(행당 1개) ②모든 행에 사진 1장+ ③사진은 저녁/야경만("목요일 저녁 8시" 정체성) ④인용카드 흰 배경 고정 ⑤캐러셀 마지막 장 CTA 고정.

## 3. 포맷 믹스

| 포맷 | 주간 | 역할 |
|---|---|---|
| 릴스 | 2 | **도달** — 비팔로워 발견의 유일한 창구 |
| 피드 | 3 | **신뢰** — 프로필 방문자의 랜딩 페이지 |
| 스토리 | 매일 1~3 | **전환·유지** — 링크 스티커 신청 직행 |

신규 계정 현실: 피드는 팔로워에게만 감 → **도달은 릴스가 전부, 피드는 진열대**.

**릴스**: 7~15초, 첫 1초 텍스트 훅, 매주 목요일 현장 5컷(집결-인사-출발-발맞춤-완주)으로 주 2편. **위치 태그 "여의도한강공원" 전 게시물 필수.** 캡션 첫 줄에 검색 키워드 문장형("여의도 한강 퇴근런, 목요일 저녁 8시 소그룹 러닝") — 지금 인스타는 해시태그보다 캡션 SEO. 주 2회×8주 채우기 전 성과 판단 금지(초기 조회 200~500이 정상).

**도달 부스터 2가지**: ①**공동 게시(Collab)** — 완주 사진·릴스를 참가자 공동 작성자로 초대 → 참가자 팔로워 피드에 노출. 매주 목요일 1개는 반드시 공동 게시. ②**참가자 스토리 리그램 루프**.

**스토리 루틴**: 월 신청 오픈+링크 / 수 "N명 확정"+마감 / 목 낮 날씨·밤 비하인드 / 금 리그램 몰이 / 일 다음 주 예고.

## 5. UGC 운영 SOP (주간)

전제: 신청 폼에 사진 사용 동의 체크박스 추가.

- **수요일**: 매칭 확정 카드 발송 + "스토리에 올리고 @ondo.pm 멘션하시면 바로 공유드릴게요" → 멘션 12시간 내 리그램
- **목요일**: 전후 10분 촬영(단체 1컷 필수) → **당일 밤 참가자에게 원본 먼저 전달** — "사진 드려요"가 먼저, 태그 요청은 그 다음 (선물 먼저 = 태그율 상승)
- **금요일**: 멘션 전부 리그램(오렌지 프레임/"THU 20:00" 텍스트로 브랜드화) + 베스트 컷 주인공에게 공동 게시 요청 (부담 없는 거절 안내 포함)
- **토~일**: 후기 한 줄 → 인용카드 제작 → 다음 주 그리드 / "후기" 하이라이트 추가 / 지표 기록

원칙: 리그램 12시간 내(환대 체감) / 인용은 개별 허락 + "3주차 참가자 J님" 표기 / 요청은 회당 1개만.
