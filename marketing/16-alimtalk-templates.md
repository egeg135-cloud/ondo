# 알림톡 템플릿 심사 제출용 문안 (카카오 비즈니스 채널 + Solapi)

**제출 순서**: 카카오 비즈니스 채널 개설 → 채널 검수 통과 → Solapi(또는 알리고)에서 발신프로필 연동 → 아래 템플릿 등록·심사 (승인 1~2주).
**심사 통과 요령**: 알림톡은 **정보성만 승인**됨 — 수신자의 신청/거래에 따른 안내여야 하고, 홍보 문구("무료!", "지금 신청하세요")가 섞이면 반려. 아래 5종은 전부 정보성으로 다듬어둠. 변수는 `#{변수명}` 형식.

## T1. 신청 접수 + 보증금 안내 (트리거: 신청 완료)

```
[ONDO] #{이름}님, 신청이 접수되었습니다.

▶ 일정: #{세션일자} (목) 저녁 8시
▶ 장소: 여의도한강공원 (여의나루역 2번 출구)
▶ 페이스 그룹: #{페이스}

보증금 #{보증금액} 입금 확인 후 매칭이 확정됩니다.
- 입금 계좌: #{은행} #{계좌번호} (예금주: #{예금주})
- 입금자명: #{이름}
- 입금 기한: #{마감일시}

보증금은 참석 시 다음 회차로 이월되며, 참여 취소 시 전액 환급됩니다. 당일 불참 시에만 소멸됩니다.
```
버튼: [신청 내역 확인] 웹링크 → https://ondo-match.vercel.app/

## T2. 매칭 확정 안내 (트리거: 어드민 확정 토글)

```
[ONDO] #{이름}님, 매칭이 확정되었습니다.

▶ 일정: #{세션일자} (목) 저녁 8시
▶ 집합: 여의나루역 2번 출구 앞 (19:40까지)
▶ 페이스 그룹: #{페이스}

#{공지사항}

참여가 어려워지신 경우 전일 23:59까지 회신 주시면 보증금이 다음 회차로 이월됩니다.
```
버튼: [오픈채팅 입장] 웹링크 → #{오픈채팅링크}

## T3. 전일 리마인드 (트리거: 수요일 18시 크론, 확정자)

```
[ONDO] #{이름}님, 내일 러닝 일정 안내드립니다.

▶ 내일(#{세션일자}, 목) 저녁 8시, 여의도한강공원 5km
▶ 집합: 여의나루역 2번 출구 앞, 19:40까지
▶ 페이스 그룹: #{페이스}

몸 상태가 좋지 않으시면 무리하지 마시고 오늘 중 회신 주세요. 건강 사유 취소 시 보증금은 이월됩니다.
```

## T4. 당일 리마인드 (트리거: 목요일 14시 크론, 확정자)

```
[ONDO] #{이름}님, 오늘 저녁 8시 러닝 일정 안내드립니다.

▶ 집합: 여의나루역 2번 출구 앞, 19:40까지
▶ 우천 등 취소 사유 발생 시 이 채널로 즉시 안내드립니다.

참석이 어려우신 경우 미리 회신 부탁드립니다.
```

## T5. 참석 확인 + 보증금 이월 안내 (트리거: 금요일 10시 크론, 참석자)

```
[ONDO] #{이름}님, 어제 5km 완주를 확인했습니다.

보증금 #{보증금액}은 다음 회차로 자동 이월되어, 다음 주 신청 시 추가 입금 없이 참여하실 수 있습니다.

이월된 보증금의 환급을 원하시면 이 채널로 회신 주세요. 3영업일 내 처리됩니다.
```
버튼: [다음 주 신청] 웹링크 → https://ondo-match.vercel.app/?apply=1&utm_source=kakao&utm_medium=notice&utm_campaign=rejoin

## ⚠️ 알림톡으로 보내면 안 되는 것 (친구톡/SMS로)

- 재참여 권유("이번 주도 뛸까요?"), 리드 리마인드("신청서 미완료"), 이벤트·행사 홍보 → **광고성**이라 알림톡 반려 대상
- 광고성은 친구톡(채널 친구 대상) 또는 SMS로: `(광고)` 표기 + 무료수신거부 안내 + 21~08시 발송 금지 + 마케팅 동의자만 (notice.ts의 문안이 이 형식으로 준비돼 있음)
