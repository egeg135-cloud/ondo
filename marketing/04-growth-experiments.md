- **A2. 자체 오픈카톡방 개설 (7.3)** — "여의도 러닝·퇴근 러닝" 키워드를 방 제목·해시태그에 박은 "ONDO 목요런 대기실". 오픈카톡 검색 유입 자동 축적, 입장자 = 리드 풀. 세팅 1h, 주 15분.
- **A3. 스트라바 클럽 + 세그먼트 (6.3)** — 참석자 업로드 시 클럽명이 러너 네트워크에 자동 노출. "기록 인증하면 조 편성 때 페이스 반영" 명분.
- **A4. /course SEO 페이지 (5.3)** — 프리렌더링 선행. "여의도 러닝 코스" 검색 상시 유입구.
- **A5. "이번 주 참여 회사" 익명 집계 노출 (6.0)** — 폼에 회사명(선택) → "증권사 2·핀테크 1" 노출. 여의도 동질감 + 블라인드 자생 언급 소재.

## 활성화 5개

- **B1. 토스 원탭 송금 링크 (8.3) ← 최우선** — 계좌복사→은행앱 마찰 제거. 금액 1만원 고정 링크. 성공 오버레이·확정 안내에 버튼, 계좌 복사는 폴백. deposit_start에 method 파라미터.
- **B2. 미입금 3시간 리마인드 (7.7)** — 어드민 "미입금 N시간" 필터 + 원클릭 템플릿. "자리 잡아두었어요, 입금 확인되면 조 편성 들어가요."
- **B3. "입금 순 페이스 조 우선 배정" (7.0)** — 실제 운영 규칙으로 만들면 거짓 없는 즉시 입금 유인.
- **B4. 동반 신청 = 같은 조 보장 (7.0)** — 폼에 친구 연락처(선택) 1칸. 동반 쌍은 노쇼 급감 + 완료 화면 "친구에게 링크 보내면 같은 조" 추천 트리거.
- **B5. 첫 참석 불안 제거 키트 (7.7)** — 당일 오후 5시: 집결 지점 실사진(화살표) + "도착하면 이렇게" 3단계 + 운영자 복장. 1회 제작 2h, 매주 5분.

## 리텐션·추천 5개

- **C1. 완주 카드 루프 (7.3) ← 최우선** — 감정 피크는 완주 직후. 단체사진+조 이름+날짜 카드를 목요일 밤 10시 전 조별 발송. URL·QR 각인, "@ondo 태그 시 리그램". shareCard.ts 템플릿 변형 반나절.
- **C2. 스트릭 리워드 (6.3)** — 4주 연속 = 키링+스트릭 카드. attended 컬럼 선행.
- **C3. 조 단위 재신청 링크 (7.0)** — "지난주 A조, 이번 주도 같은 조로?" + ?apply=1&crew=A. C1과 한 메시지로 묶으면 운영비용 0.
- **C4. 게스트 패스 (7.0)** — 참석자에게 "친구 1명 보증금 면제 초대권". 무료 서비스의 유일한 마찰(1만원)을 면제 = 추천 인센티브. 주당 상한 5장. 게스트→익주 정식 전환율이 K-factor 실측값.
- **C5. 먼슬리 랩드 (5.3)** — "누적 20km · 4번의 목요일 · 함께 뛴 러너 11명" 개인 요약 카드. attended 4주치 축적 후.

## 바이럴 루프 설계

```
     ┌────────────── 이월 보증금 (발화자 재장전) ──────────────┐
     │                                                        │
[1] 목요일 참석·완주 → [2] 완주 카드 수신 → [3] 스토리·카톡 공유
     ▲                                                        ▼
[7] 입금·확정 ← [6] 신청(동반/게스트) ← [5] 랜딩 ← [4] 친구가 카드를 봄
```

| 이탈 지점 | 보강책 |
|---|---|
| 카드 전달 지연 (감정 식음) | 목요일 밤 10시 전 발송을 운영 루틴으로 고정 |
| 공유할 이유 없음 | 본인 성과+조 이름으로 자기표현화, 태그 시 리그램, 카톡 조 단톡 공유 병행 |
| **스토리 이미지엔 링크 없음 (최대 구멍)** | 카드에 URL·QR 각인, 링크 스티커 안내 1줄, 카톡은 OG 카드가 곧 링크 |
| 클릭 안 함 | 카드 문구를 관찰자 CTA로: "다음 목요일, 이 조에 한 자리 있어요" |
| 보증금 기대위반 | share 유입 전용 배너 + 동반 신청/게스트 패스로 우회 |
| 입금 마찰 | B1+B2+B3 |
| 첫 참석 노쇼 | B5 + 동반 조 보장 |
| 발화자 이탈 | 이월 확정 메시지 + 조 단위 재신청(C3) |

**현실 목표**: 참석 20명 × 공유율 30% × 카드당 유입 2 × 전환 25% = 주당 신규 3명, K≈0.15. K>1은 비현실 — "매주 복리 루프"로 운영. **계측 선행: share_card_click + UTM 없으면 장님 운영.**

## 첫 2주 실행 3개 (B1+B2+B3 동승 / C1 / A1)

**실험 1 — 토스 원탭 (B1)**: D1 링크 발급+버튼 추가+method 파라미터+입금 순 배정 1줄 → D2 어드민 미입금 필터+템플릿 → 매일 리마인드 발송 → 2주 후 신청→입금 전환율·소요시간 판정.

**실험 2 — 완주 카드 (C1)**: 선행 share_card_click+UTM → 템플릿 변형(사진 슬롯+QR) → 목요일 해산 전 조별 단체사진 → 밤 10시 전 발송(태그 안내+재신청 링크) → 금요일 리그램 → 2회차 후 업로드율·유입·재신청률 판정.

**실험 3 — 스레드 (A1)**: 계정 개설+선언글 → 주 4회 루틴(월 숫자/수 마감/목 현장/금 비하인드) → 4주 후 UTM 신청 5건 기준 판정.

2주 총 부하: 세팅 2일 + 주간 1.5h.
