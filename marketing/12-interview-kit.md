# ONDO 유료 결제자 인터뷰 킷

**목적**: ①무료화 반응 확인 ②지불의사의 원천(=진짜 가치제안) 발굴 ③오프라인 행사에서 뭘 원하는지 사전 조사
**대상**: 유료 결제 이력자 전원 (어드민에서 `보증금 ✓` + 구 4주권 뱃지 기준 — 돈을 냈던 유일한 코호트, 5명 이상이면 패턴 신뢰 가능)
**형식 우선순위**: 러닝 동반(최고 — 자연스럽고 방어 없음) > 대면 커피 > 전화 10분

## 1. 섭외

- 환불 안내 카톡에 이미 인터뷰 요청이 포함돼 있음 ("10분 시간 내주시면 다음 세션 우선 매칭"). 응답 없는 분께 개별 팔로업:

> ○○님, 환불 처리 완료됐어요! 하나만 더 — 이번 주 목요일에 나오시면 제가 같은 조로 뛰면서 온도 얘기 좀 여쭤봐도 될까요? 인터뷰라기보다 그냥 뛰면서 수다예요 🏃 (참가비는 이제 무료입니다)

- 러닝 참석이 어려운 분: "전화 10분이면 충분해요. 편한 시간 2개만 알려주세요."

## 2. 질문지

### 러닝 동반 버전 (뛰면서 — 외우기 쉽게 4개만)

1. "원래 어쩌다 신청하셨어요? 그때 뭘 기대하셨어요?" *(지불 이유 → 진짜 가치제안)*
2. "무료로 바뀐 거 들었을 때 솔직히 어떠셨어요? 좋다? 뭔가 수상하다?" *(무료화 커뮤니케이션 방향)*
3. "온도가 뭘 해주면 '이건 돈 내고라도 쓴다' 싶으세요?" *(재수익화 씨앗)*
4. "9월에 러닝 말고 좀 큰 모임을 한 번 열어볼까 하는데 — 어떤 자리면 오고 싶으세요? 뒤풀이 없는 게 온도 스타일인데, 행사라면 어때야 할까요?" *(행사 컨셉 검증 — 핵심)*

### 전화/대면 풀 버전 (10분, 7문항)

1. 유료였는데도 결제하셨잖아요 — 그때 뭘 기대하고 결제 버튼을 누르셨어요?
2. 무료 전환 소식 첫 느낌이 어떠셨어요? (이득이다 vs 품질·지속성 걱정)
3. 무료가 되면 모임에서 뭐가 달라질 것 같으세요? 사람들, 분위기, 노쇼 — 뭐든요.
4. 보증금 1만원 방식(참석하면 이월·노쇼면 소멸·그만두면 환급) 들으면 어떤 생각이 드세요? 걸리는 부분은요?
5. (참석 경험자) 모임에서 가장 좋았던 순간 하나, 아쉬웠던 순간 하나만요. / (미참석자) 결제까지 하고 왜 못 나오셨어요?
6. 다음 주에 또 나가게 만드는 것 하나, 안 나가게 만드는 것 하나를 꼽는다면?
7. 9월에 오프라인 행사를 한 번 열려고 해요. 어떤 자리면 참석하시겠어요? — 규모(10명 vs 50명), 형식(기록회/파티/네트워킹), 유료여도 올 것 같은 가격은?

**진행 팁**: 2번을 4번보다 먼저(규칙 설명이 반응을 오염시키지 않게) · 답마다 "왜요?" 한 번만 더 파기 · 인용문 그대로 받아 적기(VOC가 곧 카피 자산) · 유도 질문 금지("좋았죠?" ❌)

## 3. 기록 템플릿 (인터뷰당 1개, Notion 회의록에)

```
## 인터뷰 — [익명코드: 유료A] (형식: 러닝동반/전화/대면, 날짜: )
- 결제 이유(원문 인용):
- 무료화 반응:
- 보증금 인식:
- 최고의 순간 / 아쉬운 순간:
- 재참여 드라이버 / 킬러:
- 유료 의사 ("~면 돈 낸다"):
- 행사 반응 (형식/규모/가격):
- 그대로 쓸 만한 문장(카피 후보):
```

## 4. 인터뷰 후 처리 루프

- 3건 모이면: 공통 패턴 1페이지 정리 → ①랜딩 카피에 VOC 문장 반영 ②행사 컨셉 결정 근거로 사용 ③Notion 회의록에 "인터뷰 종합" 기록
- 좋은 문장은 (동의 받고) 후기로 랜딩·인스타 재사용
- 인터뷰 응한 분 = 행사 1차 초대 명단 (가장 애정 있는 코호트)
