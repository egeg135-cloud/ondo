# 사업화지원금 집행 계획 (잔액 180만원 기준)

**전제**: 배정 잔액 180만원 = 지원 가능 공급가액 한도. 부가세 10% 자부담 → 실제 선지출 198만원, 후지급으로 180만원 회수. 123스튜디오 무료 촬영 1회 제공분은 중복지원 금지 조항 때문에 촬영비 지원 신청 불가 → 촬영은 스튜디오, 지원금은 편집·광고·인쇄·상표로.

- 전 건 총액 100만원 이하 → **비교견적서 전부 면제**
- ⚠️ 편집 외주는 집행 전 센터 확인 필수: "스튜디오 촬영 지원과 별개로 편집 용역비 집행 가능한가요?" → 불가 시 40만원을 네이버 광고로 이관 (양식2 변경보고서)
- ⚠️ 크몽·숨고 금지 — 편집·인쇄 모두 업체 직접 계약 (세금계산서 발행 업체)
- ⚠️ 사용계획서에 위 4개 항목이 없으면 집행 전에 양식2 변경보고서 먼저

- [ ] (전자)세금계산서 + 이체확인증(계좌 마스킹 금지) 또는 카드영수증
- [ ] 거래명세서 / 견적서
- [ ] 거래처 사업자등록증·통장사본 (계약 시점에 미리 요청 — 업태·업종이 과업과 맞는지 확인)

- [ ] 검수조서(양식5, 결과물 사진 필수) + 결과보고서(양식4 — 광고는 GA4 UTM 데이터로 "데이터 기반" 요건 충족)
- [ ] 광고: 게시물·집행기간 내역 캡처 / 영상: 원본 파일 / 인쇄: 실물 사진
- [ ] 상표: 출원사실증명원·출원번호통지서 (출원인 = 대표자 본인 명의 필수)
- [ ] 우리 사업자등록증·통장사본

## 중복지원 관리

- 123스튜디오 촬영 = 센터 제공 지원 → 동일 건(촬영비) 지원금 신청 금지
- 타 정부·지자체 지원사업과 같은 지출 건 중복 신청 시 환수 — 다른 사업 집행 내역과 항목 대조 후 신청
