## 1. 온도의 현재 상태 요약

- **상태**: **진행 중** (7/17 무료 전환 재런칭 직후 — 검증 사이클 재시작 단계)
- **지금까지 한 것**:
  - 신청 웹앱 구축·운영 (ondo-match.vercel.app: 페이스 선택 → 5단계 신청 → 보증금 안내, 어드민 대시보드, 카톡 안내문 자동 생성)
  - 6~7월 유료 파일럿 (체험 5,000원/4주 19,000원) — 신청 12명, 세션 수 회 진행, 후기 5건 확보
  - 7/15 전면 무료화 + 보증금 1만원 이월형 전환, 기존 결제자 환불 진행
  - SEO/기술 정비 완료 (프리렌더링·JSON-LD·OG이미지·GA4 이벤트 체계·참석 기록 기능)
  - 마케팅 실행 자산 일체 제작 완료 (카드뉴스 4세트 28장, 4주 캘린더, 채널별 씨딩 스크립트, 콘텐츠 팩)
- **남아 있는 자산**: 신청자 12명 DB + 리드(전화번호) 리스트, 후기 5건, 인스타 @ondo.pm + 카드뉴스 28장, 운영 노하우(매칭·카톡 안내 프로세스), 여의도 5km 코스, 랜딩+어드민 코드베이스, GA4/Supabase 측정 인프라, 마케팅 플레이북 문서 12개(리포지토리 marketing/ 폴더)
- **현재 가장 큰 문제**: **신규 유입 절대량 부족** — 제품·콘텐츠·측정은 준비됐으나 채널 실행(당근·인스타·오픈카톡)이 아직 0회. 주당 신규 신청 15건이 필요한데 현재 주 1~2건 수준
- **가장 먼저 검증해야 할 것**: ① 무료+보증금 모델에서 **신청→입금(보증금) 전환율** — 무료화가 허수 신청만 늘리는지, 진짜 러너를 데려오는지 ② **재참여율 50%** — 이월 보증금 구조가 실제로 재방문을 만드는지 (이 둘이 무너지면 100명 목표 자체가 불가)

## 10. 기타 참고사항

- **담당자**: 오동호 (1인 운영 — 개발·마케팅·현장 운영 겸임, 기술 작업은 Claude 협업)
- **현재 가장 급한 일** (이번 주 내, 순서대로):
  1. 카카오 비즈니스 채널 개설 + 알림톡 템플릿 심사 신청 (승인 1~2주 리드타임)
  2. 인스타 세트1 게시 + 당근 3개 동네 모임 개설 (콘텐츠 전부 준비됨)
  3. 네이버 서치어드바이저 등록 (프리렌더링 7/19 배포 완료 — 지금이 최적기)
  4. 리드 문자 발송은 법무 체크 통과 전 금지 (동의 없이 수집된 기존 리드는 발송 불가)
- **대표가 확인해야 할 부분**:
  1. 보증금 수취용 사업자 계좌 개설 여부·시점 (현재 개인 카카오뱅크)
  2. 소멸 보증금의 소득 처리 — 세무사 1회 확인
  3. 이용약관·개인정보처리방침 개정안 최종 승인 (특히 노쇼 소멸 조항)
  4. 온도 vs 리듬 리소스 배분 — 캠페인 주 6~8시간, P1 최소셋만이면 주 4시간
- **필요한 지원**:
  - 토스 익명 송금 링크 발급 (1만원 고정) — 발급 즉시 "1초 송금" 버튼 활성화 가능 (코드 준비 완료)
  - 목요일 세션 촬영 보조 1인 (격주라도)
  - 알림톡 발송비 예산: 월 약 1.5만원 (100명 기준)
